# Architecture

Agents specialized in system architecture and solution design. Focuses on
scalability, design patterns, and architectural decisions.

## Agents

This plugin provides the following specialized agents:

- **Solution Architect**
- **System Architect**

## Usage

Agents can be invoked using the Task tool with the appropriate subagent_type or through
natural conversation:

```text
Use the Solution Architect to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

```bash
han plugin install architecture
```
