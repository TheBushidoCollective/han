# Blockchain

Blockchain development agents providing expertise in blockchain architecture,
smart contracts, and Web3

## Agents

This plugin provides the following specialized agents:

- **Blockchain Engineer**
- **Smart Contract Engineer**
- **Web3 Engineer**

## Usage

Agents can be invoked using the Task tool with the appropriate subagent_type or through
natural conversation:

```text
Use the Blockchain Engineer to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

```bash
han plugin install blockchain
```
