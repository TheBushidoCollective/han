# Infrastructure

Agents specialized in infrastructure management, DevOps, and site reliability
engineering. Focuses on deployment, monitoring, and system operations.

## Agents

This plugin provides the following specialized agents:

- **Devops Engineer**
- **Site Reliability Engineer**

## Usage

Agents can be invoked using the Task tool with the appropriate subagent_type or through
natural conversation:

```text
Use the Devops Engineer to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

```bash
han plugin install infrastructure
```
