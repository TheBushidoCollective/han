# Data Engineering

Data engineering agents providing expertise in ETL pipelines, streaming, and
data warehousing

## Agents

This plugin provides the following specialized agents:

- **Data Pipeline Engineer**
- **Data Warehouse Engineer**
- **Streaming Engineer**

## Usage

Agents can be invoked using the Task tool with the appropriate subagent_type or through
natural conversation:

```text
Use the Data Pipeline Engineer to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

```bash
han plugin install data-engineering
```
