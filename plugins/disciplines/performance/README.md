# Performance

Performance engineering agents providing expertise in profiling, optimization,
and scalability

## Agents

This plugin provides the following specialized agents:

- **Optimization Engineer**
- **Performance Engineer**
- **Scalability Engineer**

## Usage

Agents can be invoked using the Task tool with the appropriate subagent_type or through
natural conversation:

```text
Use the Optimization Engineer to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

```bash
han plugin install performance
```
