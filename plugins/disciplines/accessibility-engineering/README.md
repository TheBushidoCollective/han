# Accessibility

Accessibility engineering agents providing expertise in WCAG, ARIA, and
inclusive design

## Agents

This plugin provides the following specialized agents:

- **Accessibility Engineer**
- **Assistive Tech Specialist**
- **Inclusive Design Engineer**

## Usage

Agents can be invoked using the Task tool with the appropriate subagent_type or through
natural conversation:

```text
Use the Accessibility Engineer to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

```bash
han plugin install accessibility
```
