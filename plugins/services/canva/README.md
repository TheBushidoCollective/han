# Canva

Canva design platform integration via the official Canva MCP server.

## Features

- **Create designs** - Generate new designs from templates or scratch
- **Auto-fill templates** - Populate Canva templates with your content
- **Search designs** - Find existing designs in your Canva account
- **Export** - Download designs as PDF, PNG, or other formats

## Authentication

This plugin uses OAuth authentication handled automatically by Claude Code. When you first use Canva tools, you'll be prompted to connect your Canva account.

## Example Usage

```
Create a social media post announcing our new feature launch
```

```
Export my latest presentation as a PDF
```

```
Find designs I created last week with our company logo
```

## Requirements

- Canva account (free tier works)
- Claude Code with MCP support
