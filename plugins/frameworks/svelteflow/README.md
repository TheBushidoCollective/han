# Svelte Flow

Build node-based editors, interactive diagrams, and workflow builders with Svelte Flow.

## Installation

```bash
han plugin install svelteflow
```

## What's Included

- Skills for building node-based UIs, custom nodes, edges, and layouts
- Best practices for Svelte Flow component architecture
- Guidance on interactive diagram and workflow builder patterns
