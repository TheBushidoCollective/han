# Specialized Plugins

This directory contains plugins for specialized or niche domains that don't fit neatly into the other categories.

## Contents

| Plugin | Description |
|--------|-------------|
| android | Android development with Gradle and Jetpack Compose |
| claude-agent-sdk | Claude Agent SDK for building AI agents |
| fnox | Fnox secrets management |
| han-plugins | Han plugin structure validation |
| ios | iOS development with Xcode |
| notetaker | Code annotation and note-taking patterns |
| runbooks | Operational runbook documentation |
| scratch | Temporary workspace enforcement |
| sentry | Sentry error monitoring patterns |
| sip | Session Initiation Protocol for VoIP |
| tensorflow | TensorFlow machine learning |

## Category Guidelines

Specialized plugins typically:

- Target niche domains or ecosystems
- Have limited overlap with mainstream development
- May require specialized knowledge or tooling
- Are often platform-specific or industry-specific
