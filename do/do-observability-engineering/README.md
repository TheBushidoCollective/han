# Do Observability Engineering

Observability engineering agents providing expertise in tracing, monitoring, and
logging

## Agents

This plugin provides the following specialized agents:

- **Logging Engineer**
- **Monitoring Engineer**
- **Observability Engineer**

## Usage

Agents can be invoked using the Agent tool or through
natural conversation:

```text
Use the Logging Engineer to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

For `han` CLI installation instructions, visit [han.guru](https://han.guru).

```bash
han install do-observability-engineering
```
