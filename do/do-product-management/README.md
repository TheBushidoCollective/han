# Do Product Management

Agents specialized in product vision and strategy. Focuses on product direction,
market positioning, and strategic planning.

## Agents

This plugin provides the following specialized agents:

- **The Visionary**

## Usage

Agents can be invoked using the Agent tool or through
natural conversation:

```text
Use the The Visionary to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

For `han` CLI installation instructions, visit [han.guru](https://han.guru).

```bash
han install do-product-management
```
