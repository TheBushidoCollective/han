# Do Graphics Engineering

Graphics engineering agents providing expertise in GPU programming, shaders, and
rendering

## Agents

This plugin provides the following specialized agents:

- **Graphics Engineer**
- **Shader Engineer**
- **Visualization Engineer**

## Usage

Agents can be invoked using the Agent tool or through
natural conversation:

```text
Use the Graphics Engineer to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

Install with npx (no installation required):

```bash
npx @thebushidocollective/han plugin install do-graphics-engineering
```
