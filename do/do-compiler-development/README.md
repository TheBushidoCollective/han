# Do Compiler Development

Compiler development agents providing expertise in lexing, parsing, code
generation, and language design

## Agents

This plugin provides the following specialized agents:

- **Compiler Engineer**
- **Language Designer**
- **Vm Engineer**

## Usage

Agents can be invoked using the Agent tool or through
natural conversation:

```text
Use the Compiler Engineer to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

Install with npx (no installation required):

```bash
npx @thebushidocollective/han plugin install do-compiler-development
```
