# Do Database Engineering

Database engineering agents providing expertise in schema design, query
optimization, and reliability

## Agents

This plugin provides the following specialized agents:

- **Database Designer**
- **Database Reliability Engineer**
- **Query Optimizer**

## Usage

Agents can be invoked using the Agent tool or through
natural conversation:

```text
Use the Database Designer to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.
