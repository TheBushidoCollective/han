# Do Enforce Planning

Enforces planning before implementation work begins

## Agents

This plugin provides the following specialized agents:

- **Planner**

## Usage

Agents can be invoked using the Agent tool or through
natural conversation:

```text
Use the Planner to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

Install with npx (no installation required):

```bash
han plugin install do-enforce-planning
```
