# Do Project Management

Agents specialized in project coordination and technical leadership. Focuses on
workflow coordination and team dynamics.

## Agents

This plugin provides the following specialized agents:

- **Flow Coordinator**
- **Technical Coordinator**

## Usage

Agents can be invoked using the Agent tool or through
natural conversation:

```text
Use the Flow Coordinator to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

For `han` CLI installation instructions, visit [han.guru](https://han.guru).

```bash
han install do-project-management
```
