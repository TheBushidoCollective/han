# Do Machine Learning Engineering

ML engineering agents providing expertise in MLOps, model deployment, and
inference optimization

## Agents

This plugin provides the following specialized agents:

- **Ml Inference Engineer**
- **Ml Pipeline Engineer**
- **Mlops Engineer**

## Usage

Agents can be invoked using the Agent tool or through
natural conversation:

```text
Use the Ml Inference Engineer to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

Install with npx (no installation required):

```bash
han plugin install do-machine-learning-engineering
```
