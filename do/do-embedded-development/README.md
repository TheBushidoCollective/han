# Do Embedded Development

Embedded development agents providing expertise in RTOS, firmware, and IoT

## Agents

This plugin provides the following specialized agents:

- **Embedded Engineer**
- **Firmware Engineer**
- **Iot Engineer**

## Usage

Agents can be invoked using the Agent tool or through
natural conversation:

```text
Use the Embedded Engineer to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

Install with npx (no installation required):

```bash
npx @thebushidocollective/han plugin install do-embedded-development
```
