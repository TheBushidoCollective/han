# unit-06-navigation-redesign

## Description

Redesign site navigation to mirror SAFe/LEAN patterns with clear paths for different user journeys.

## Discipline

frontend - Navigation architecture and landing page design.

## Success Criteria

### Navigation Structure
- [ ] Main nav: Overview | Learn | Implement | Tools | Resources
- [ ] Mega menu with visual previews
- [ ] Breadcrumb navigation
- [ ] Footer with comprehensive sitemap

### Landing Page
- [ ] Hero with Big Picture preview and value proposition
- [ ] "What is AI-DLC?" section with key differentiators
- [ ] Social proof (if available - logos, testimonials)
- [ ] Three paths: "Learn", "Try It", "Implement"
- [ ] Latest blog posts/updates

### Start Here Flow
- [ ] /start-here page for new users
- [ ] Guided path through: Concepts → Mode Selection → First Intent
- [ ] Progress tracking (optional, localStorage)
- [ ] "Skip to Plugin" option for experienced users

### Mobile Navigation
- [ ] Hamburger menu with full navigation
- [ ] Bottom nav for key actions
- [ ] Swipe gestures for interactive tools

## Notes

- Reference SAFe navigation: clear taxonomy, multiple entry points
- Reference LEAN: simple, focused, educational
- Maintain existing URL structure where possible (redirects if needed)
