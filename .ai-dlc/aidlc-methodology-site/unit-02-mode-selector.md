# unit-02-mode-selector

## Description

Create an interactive decision tree tool that helps teams choose the right operating mode (HITL, OHOTL, AHOTL) based on their context.

## Discipline

frontend - Interactive wizard/questionnaire component.

## Success Criteria

- [ ] Step-by-step questionnaire with 3-5 questions
- [ ] Questions about: task complexity, team experience, risk tolerance, verification needs
- [ ] Visual progress indicator
- [ ] Recommended mode shown with confidence level
- [ ] Explanation of why that mode was recommended
- [ ] Links to detailed mode documentation
- [ ] "Start over" and "Back" navigation
- [ ] Shareable results URL

## Notes

- Questions should be based on the paper's mode selection criteria
- Consider showing a comparison table at the end
- Make it feel like a diagnostic tool, not a quiz
