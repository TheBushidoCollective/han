# Buki Csharp

Advanced C# programming skills for async patterns and LINQ

## Skills

This plugin provides the following skills:

- **Csharp Async Patterns**
- **Csharp Linq**
- **Csharp Nullable Types**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-csharp:csharp-async-patterns")
```

Each skill provides specialized knowledge and patterns
for Csharp development.
