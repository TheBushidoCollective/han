# Buki Apollo Graphql

Advanced Apollo GraphQL skills for client and server development

## Skills

This plugin provides the following skills:

- **Apollo Caching Strategies**
- **Apollo Client Patterns**
- **Apollo Server Patterns**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-apollo-graphql:apollo-caching-strategies")
```

Each skill provides specialized knowledge and patterns
for Apollo Graphql development.
