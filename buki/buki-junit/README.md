# Buki Junit

Comprehensive JUnit testing framework skills for Java projects with annotations,
assertions, and extensions.

## Skills

This plugin provides the following skills:

- **Junit Extensions**
- **Junit Fundamentals**
- **Junit Parameterized**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-junit:junit-extensions")
```

Each skill provides specialized knowledge and patterns
for Junit development.
