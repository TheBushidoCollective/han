# Buki Rust

Advanced Rust skills for ownership, error handling, and async programming.

## Skills

This plugin provides the following skills:

- **Async Patterns**
- **Error Handling**
- **Ownership System**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-rust:rust-async-patterns")
```

Each skill provides specialized knowledge and patterns
for Rust development.
