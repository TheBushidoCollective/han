# Buki Django

Advanced Django skills for ORM, class-based views, and REST framework.

## Skills

This plugin provides the following skills:

- **Django Cbv Patterns**
- **Django Orm Patterns**
- **Django Rest Framework**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-django:django-cbv-patterns")
```

Each skill provides specialized knowledge and patterns
for Django development.
