# Buki Clippy

Comprehensive Clippy skills for Rust linting and code quality with lint
configuration, categories, and custom lint development.

## Skills

This plugin provides the following skills:

- **Clippy Configuration**
- **Clippy Custom**
- **Clippy Lints**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-clippy:clippy-configuration")
```

Each skill provides specialized knowledge and patterns
for Clippy development.
