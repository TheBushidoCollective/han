# Buki C

Advanced C programming skills for systems programming and memory management

## Skills

This plugin provides the following skills:

- **C Data Structures**
- **C Memory Management**
- **C Systems Programming**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-c:c-data-structures")
```

Each skill provides specialized knowledge and patterns
for C development.
