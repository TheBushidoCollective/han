# Buki Rails

Advanced Ruby on Rails skills for MVC patterns, Active Record, and Hotwire

## Skills

This plugin provides the following skills:

- **Rails Action Controller Patterns**
- **Rails Active Record Patterns**
- **Rails Hotwire**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-rails:rails-action-controller-patterns")
```

Each skill provides specialized knowledge and patterns
for Rails development.
