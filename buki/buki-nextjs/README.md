# Buki Nextjs

Advanced Next.js skills for App Router, Server Components, and data fetching.

## Skills

This plugin provides the following skills:

- **Nextjs App Router**
- **Nextjs Data Fetching**
- **Nextjs Server Components**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-nextjs:nextjs-app-router")
```

Each skill provides specialized knowledge and patterns
for Nextjs development.
