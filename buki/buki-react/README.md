# Buki React

Advanced React skills for hooks, context, and performance optimization.

## Skills

This plugin provides the following skills:

- **React Context Patterns**
- **React Hooks Patterns**
- **React Performance**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-react:react-context-patterns")
```

Each skill provides specialized knowledge and patterns
for React development.
