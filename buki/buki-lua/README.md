# Buki Lua

Advanced Lua programming skills for embedded scripting and game development

## Skills

This plugin provides the following skills:

- **Lua C Integration**
- **Lua Coroutines**
- **Lua Tables Patterns**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-lua:lua-c-integration")
```

Each skill provides specialized knowledge and patterns
for Lua development.
