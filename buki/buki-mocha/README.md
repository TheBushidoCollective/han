# Buki Mocha

Comprehensive Mocha testing framework skills for JavaScript and Node.js projects
with flexible assertion libraries.

## Skills

This plugin provides the following skills:

- **Mocha Assertions**
- **Mocha Configuration**
- **Mocha Fundamentals**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-mocha:mocha-assertions")
```

Each skill provides specialized knowledge and patterns
for Mocha development.
