# Buki Php

Advanced PHP programming skills for modern PHP and security patterns

## Skills

This plugin provides the following skills:

- **Php Composer Autoloading**
- **Php Modern Features**
- **Php Security Patterns**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-php:php-composer-autoloading")
```

Each skill provides specialized knowledge and patterns
for Php development.
