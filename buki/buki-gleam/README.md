# Buki Gleam

Advanced Gleam programming skills for functional BEAM development

## Skills

This plugin provides the following skills:

- **Gleam Actor Model**
- **Gleam Interop**
- **Gleam Type System**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-gleam:gleam-actor-model")
```

Each skill provides specialized knowledge and patterns
for Gleam development.
