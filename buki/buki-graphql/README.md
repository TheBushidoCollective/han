# Buki Graphql

Advanced GraphQL skills for schema design, resolvers, and query optimization.

## Skills

This plugin provides the following skills:

- **Graphql Performance**
- **Graphql Resolvers**
- **Graphql Schema Design**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-graphql:graphql-performance")
```

Each skill provides specialized knowledge and patterns
for Graphql development.
