# Buki Go

Advanced Go skills for concurrency, error handling, and interfaces.

## Skills

This plugin provides the following skills:

- **Concurrency**
- **Error Handling**
- **Interfaces**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-go:go-concurrency")
```

Each skill provides specialized knowledge and patterns
for Go development.
