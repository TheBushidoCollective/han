# Buki Kotlin

Advanced Kotlin programming skills for coroutines and DSL patterns

## Skills

This plugin provides the following skills:

- **Kotlin Coroutines**
- **Kotlin Dsl Patterns**
- **Kotlin Null Safety**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-kotlin:kotlin-coroutines")
```

Each skill provides specialized knowledge and patterns
for Kotlin development.
