# Buki Vitest

Comprehensive Vitest testing framework skills for modern Vite-powered JavaScript
and TypeScript projects.

## Skills

This plugin provides the following skills:

- **Vitest Configuration**
- **Vitest Performance**
- **Vitest Testing Patterns**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-vitest:vitest-configuration")
```

Each skill provides specialized knowledge and patterns
for Vitest development.
