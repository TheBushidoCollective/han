# Buki Python

Advanced Python skills for type system, async patterns, and data modeling.

## Skills

This plugin provides the following skills:

- **Async Patterns**
- **Data Classes**
- **Type System**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-python:python-async-patterns")
```

Each skill provides specialized knowledge and patterns
for Python development.
