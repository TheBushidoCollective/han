# Buki Cpp

Advanced C++ programming skills for modern C++ and template metaprogramming

## Skills

This plugin provides the following skills:

- **Cpp Modern Features**
- **Cpp Smart Pointers**
- **Cpp Templates Metaprogramming**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-cpp:cpp-modern-features")
```

Each skill provides specialized knowledge and patterns
for Cpp development.
