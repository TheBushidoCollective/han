# Buki Testng

Comprehensive TestNG testing framework skills for Java projects with advanced
features like data providers and parallel execution.

## Skills

This plugin provides the following skills:

- **Testng Data Driven**
- **Testng Fundamentals**
- **Testng Parallel**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-testng:testng-data-driven")
```

Each skill provides specialized knowledge and patterns
for Testng development.
