# Buki Cypress

Comprehensive Cypress testing framework skills for end-to-end testing of web
applications with custom commands and CI/CD integration.

## Skills

This plugin provides the following skills:

- **Cypress Advanced**
- **Cypress Ci Cd**
- **Cypress Fundamentals**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-cypress:cypress-advanced")
```

Each skill provides specialized knowledge and patterns
for Cypress development.
