# Buki Nim

Advanced Nim programming skills for systems programming and metaprogramming

## Skills

This plugin provides the following skills:

- **Nim C Interop**
- **Nim Memory Management**
- **Nim Metaprogramming**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-nim:nim-c-interop")
```

Each skill provides specialized knowledge and patterns
for Nim development.
