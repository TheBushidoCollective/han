# Buki Objective C

Advanced Objective-C programming skills for iOS and macOS development

## Skills

This plugin provides the following skills:

- **Objc Arc Patterns**
- **Objc Blocks Gcd**
- **Objc Protocols Categories**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-objective-c:objc-arc-patterns")
```

Each skill provides specialized knowledge and patterns
for Objective C development.
