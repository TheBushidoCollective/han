# Buki Playwright

Advanced Playwright skills for end-to-end testing, test architecture, and test
automation best practices.

## Skills

This plugin provides the following skills:

- **Playwright Fixtures And Hooks**
- **Playwright Page Object Model**
- **Playwright Test Architecture**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-playwright:playwright-fixtures-and-hooks")
```

Each skill provides specialized knowledge and patterns
for Playwright development.
