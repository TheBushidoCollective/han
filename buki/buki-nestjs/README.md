# Buki Nestjs

Advanced NestJS skills for dependency injection, guards, and testing.

## Skills

This plugin provides the following skills:

- **Nestjs Dependency Injection**
- **Nestjs Guards Interceptors**
- **Nestjs Testing**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-nestjs:nestjs-dependency-injection")
```

Each skill provides specialized knowledge and patterns
for Nestjs development.
