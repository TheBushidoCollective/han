# Buki Erlang

Advanced Erlang programming skills for concurrent and distributed systems

## Skills

This plugin provides the following skills:

- **Erlang Concurrency**
- **Erlang Distribution**
- **Erlang Otp Behaviors**

## Usage

Skills can be invoked using the Skill tool:

```javascript
Skill("buki-erlang:erlang-concurrency")
```

Each skill provides specialized knowledge and patterns
for Erlang development.
